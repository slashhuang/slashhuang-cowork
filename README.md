# 个人内容工作台

> 微信公众号 & 小红书内容发布工具

**Markdown 是草稿。HTML 是读者看到的内容。**

这个工作台用 AI 把你写的文章变成：
- 📱 **小红书卡片**（1080×1440，AI 按段落智能分页，支持大字报/杂志/极简主题）
- 📰 **公众号头条封面**（900×383，支持杂志/大字标题/渐变背景）
- 📝 **文章预览**（一键复制 WeChat 兼容的 HTML，直接粘贴到公众号编辑器）

---

## 快速开始

```bash
git clone <your-repo>
cd html-anything
pnpm install
pnpm -F @html-anything/next dev
# → http://localhost:3000
```

## 预览区模式

| 模式 | 尺寸 | 用途 |
|------|------|------|
| 📝 文章 | 自适应 | 普通文章预览，可复制 HTML |
| 📱 小红书 | 1080×1440 | AI 分页卡片，多主题可选 |
| 📰 公众号封面 | 900×383 | 头条封面 |

---

## 架构

| 层 | 技术 |
|------|------|
| 前端 | Next.js 16 + React 19 + Tailwind v4 + zustand |
| Agent | 本地 CLI 检测（Claude / Codex / Cursor / Gemini / Copilot / Qwen / Aider）|
| 导出 | HTML → PNG（modern-screenshot）/ juice 内联 CSS / 下载 .html |
| 预览 | `<iframe sandbox>` 沙箱隔离 |

---

## 保留能力

- **Agent 后端** — 8 个 CLI 自动检测，SSE 流式渲染，零 API key
- **模板系统** — 75+ skill（文章/PPT/海报/视频帧等），按需扩展
- **导出** — WeChat / 知乎 / X / PNG / HTML 下载
