【模板: 像素 / 8-bit 动画解说】
【意图】教育型动画的单帧海报, 纯 CSS keyframes 循环, 不用 JS。
【布局】
- Full-bleed cream stage
- Bold display year / 大字数字
- 中心一个像素艺术 mascot (SVG 或纯 CSS 绘制)
- kinetic 中文 / 日文 display 字
- 底部 timeline ribbon 一直走
【设计细节】
- 动画用 @keyframes, 不依赖 JS
- 复古调色板: 红 / 米 / 墨绿
