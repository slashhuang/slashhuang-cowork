【模板: Simple Deck】
【意图】干净通用的 horizontal-swipe deck (pitch / overview / study)。
【布局】
- Cover + N 个 content 页 + 收尾 (N 由【用户内容】长度决定, 完整覆盖每个要点; 短内容 6-10 起步, 长内容应更多)
- 每页一个核心信息 + 1 张图 / 1 个图表
- 顶部 progress bar
【设计细节】
- 键盘 ← / → 切换 + hash 同步
