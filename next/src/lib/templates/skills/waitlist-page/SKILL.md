【模板: 等候名单页 / Waitlist】
【意图】为新产品 / 早鸟内测做一张极简等候页。
【布局】
- 居中布局: brand logo + 一行 tagline + 大字 hero (说清楚做什么)
- 邮箱捕获 input + submit 按钮 (合并成一个 pill)
- 下方 3 个小卖点 (icon + 一行字)
- 底部 founders note + 社交链接
【设计细节】
- 装饰: SVG 渐变 mesh / 噪点纹理 / 一颗星轨
- 成功提交后给一个微动效 (✓ + 文案变化)
