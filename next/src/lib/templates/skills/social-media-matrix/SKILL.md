【模板: 社媒矩阵追踪 (Social Matrix Tracker)】
【意图】电影感、数据密集的多平台社媒看板。
【布局】
- Hero header (统一品牌 + 时间窗 + 主题切换)
- Multi-platform KPI matrix (横 platform × 纵 metric)
- 交互式 charts (hover tooltip + range compare)
- Insights drawer (右抽屉, 文字洞察)
- Top posts grid
【设计细节】
- dark / light 切换; 数据用 seed 兜底
