【模板: 团队周报 Deck】
【意图】6-8 页 horizontal-swipe slides, 周报固定结构。
【布局】
- Cover (周次 + 团队 + 一句话主题)
- Shipped (列表 + owner)
- In flight (进度条)
- Blocked (红色 callout)
- Metrics (KPI 卡片网格 + 周对比)
- Asks (求助清单)
- Thanks 收尾
【设计细节】
- 键盘左右切换, hash 同步
